# Minimum Window Substring (LC#76)

Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string "".

The testcases will be generated such that the answer is unique.

## Solution

- Using sliding window (atleast required)
    - Code:
        ```cpp
        int checkWindowFreq(int charFreqWindow[52], int charFreqT[52]) {
            for(int i = 0; i < 52; ++i)
                if(charFreqWindow[i] < charFreqT[i])
                    return false;
            return true;
        }

        string minWindow(string s, string t) {
            // data required for window validation
            int charFreqT[52] = {0};
            for(char c: t) {
                if(c >= 'a' && c <= 'z')
                    charFreqT[c - 'a']++;
                else
                    charFreqT[c - 'A' + 26]++;
            }

            int n = s.length();

            // pointers
            int l = 0, r = 0;

            // state
            int charFreqWindow[52] = {0};

            // answer
            int minWindowSize = INT_MAX;
            int minWindowL = -1;
        
            while(r < n) {
                // grow
                if(s[r] >= 'a' && s[r] <= 'z')
                    charFreqWindow[s[r] - 'a']++;
                else
                    charFreqWindow[s[r] - 'A' + 26]++;
                r++;

                // shrink
                while(checkWindowFreq(charFreqWindow, charFreqT)) {
                    if(r - l < minWindowSize) {
                        minWindowSize = r - l;
                        minWindowL = l;
                    }
                    if(s[l] >= 'a' && s[l] <= 'z')
                        charFreqWindow[s[l] - 'a']--;
                    else
                        charFreqWindow[s[l] - 'A' + 26]--;
                    l++;
                }
            }

            return minWindowL == -1 ? "" : s.substr(minWindowL, minWindowSize);
        }
        ```
    - TC: $O(n)$
    - SC: $O(1)$
