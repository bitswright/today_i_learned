# Merge k Sorted Lists (LC#23)

You are given an array of k linked-lists lists, each linked-list is sorted in ascending order.

Merge all the linked-lists into one sorted linked-list and return it.

## Solution

- Using actual sorting method
    - Merge all the linked list, then apply sorting over them.
    - TC: $O(nlogn)$
    - SC: $O(1)$
- Using MinHeap:
    - Store the very elements of each list, and then keep on poping the minimum element out of the minheap and insert the next element of that list.
    - Code:
        ```cpp
        struct GreaterNode {
            bool operator()(ListNode* n1, ListNode* n2) {
                return n1->val > n2->val;
            }
        }; 

        ListNode* mergeKLists(vector<ListNode*>& lists) {
            priority_queue<ListNode*, vector<ListNode*>, GreaterNode> pq;
            for(auto list: lists) {
                if(list)
                    pq.push(list);
            }

            ListNode dummyHead(-1);
            ListNode* tail = &dummyHead;

            while(!pq.empty()) {
                ListNode* topNode = pq.top();
                pq.pop();
                tail->next = topNode;
                tail = topNode;
                ListNode* nextNode = topNode->next;
                topNode->next = NULL;
                if(nextNode)
                    pq.push(nextNode);
            }

            return dummyHead.next;
        }
        ```
    - TC: $O(nlogk)$
    - SC: $O(k)$
